== Description ==

The RestroPress - Menu Cart plugin is useful for the RestroPress users who want to show a cart menu on the header.

Available features:
- You can chnage the cart icon image by upload your required image(.png,.jpg,.jpeg,.svg)
- Setting option provided to show cart quantity with cart total price.
- Manage to show the cart icon as per the value.

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "RestroPress - Menu Cart".
3. Click install.
4. Click activate.
5. Once the plugin is installed then you can see RestroPress on the left navigation bar of WordPress Dashboard.
